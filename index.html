📘 How I Built My Online Examination Portal

Creating an Online Examination Portal was one of the most exciting and challenging projects I've worked on...

🧠 The Idea

The core concept was to create a web-based platform where students can log in and take exams...

🛠️ Tech Stack Used

  • Frontend: HTML, CSS, JavaScript, JSP
  • Backend: Java (Servlets), Maven, MySQL, Apache Tomcat

🧩 Key Features

  1. User Authentication: Session-based login.
  2. Admin Dashboard: Manage questions/exams.
  3. Timed Exams: Auto-submit with JavaScript timer.
  4. Random Questions: SQL `ORDER BY RAND()`.
  5. Score Calculation: Servlet compares answers.
